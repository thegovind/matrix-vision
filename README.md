**Prerequisites:** Node.js

### Quick Start
Run the automated startup script:
```bash
./start.sh
```

### Manual Setup
1. Install dependencies:
   `npm install`
2. Run the development server:
   `npm run dev`

The app will be available at http://localhost:3000
